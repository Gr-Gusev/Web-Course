Получение списков учающихся по курсам:
 ... -> Страница курса -> Список учащихся

Занесение учающегося в список слушателей курса:
 ... -> Страница курса -> "Добавить учащегося"
 / удаление из списка:
 ... -> Страница курса -> Список учащихся -> "Удалить учащегося"

Получение истории обучения для данного учащегося:
 ... -> Страница учащегося -> "История обучения"

Получение списков преподавателей по проводимым курсам:
 ... -> Страница курса -> Список преподавателей
 / работающих в данной компании:
 ... -> Страница компании -> Список преподавателей

Получение списков курсов, проводимых компанией:
 ... -> Страница компании -> Список проводимых курсов
 / которые читает данный преподаватель:
 ... -> Страница преподавателя -> Список читаемых курсов

Составление расписания занятий для курса:
 ... -> Страница курса -> "Расписание занятий" -> "Добавить занятие"
 ... -> Страница курса -> "Расписание занятий" -> Страница занятия -> "Изменить"
 ... -> Страница курса -> "Расписание занятий" -> Страница занятия -> "Удалить"

Просмотр, редактирование данных о занятии, удаление его:
 ... -> Страница занятия
 ... -> Страница занятия -> "Изменить"
 ... -> Страница занятия -> "Удалить"
 
Получение расписания на заданный интервал времени для обучающегося, преподавателя:
 ... -> Страница учащегося -> "Расписание занятий" -> Применение фильтров по дате
 ... -> Страница преподавателя -> "Расписание занятий" -> Применение фильтров по дате

Добавление и удаление курса, чтение и редактирование данных о нём:
 Домашняя страница -> "Курсы" -> "Добавить"
 ... -> Страница курса -> "Изменить"
 ... -> Страница курса -> "Удалить"

Добавление и удаление обучающихся и преподавателей, чтение и редактирование данных о них:
 Домашняя страница -> "Преподаватели" ("Учащиеся") -> "Добавить"
 ... -> Страница преподавателя (учащегося) -> "Изменить"
 ... -> Страница преподавателя (учащегося) -> "Удалить"

Добавление и удаление компании, чтение и редактирование данных о ней:
 Домашняя страница -> "Компании" -> "Добавить"
 ... -> Страница компании -> "Изменить"
 ... -> Страница компании -> "Удалить"
