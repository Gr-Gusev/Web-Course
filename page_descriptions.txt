Определение набора страниц, навигации между ними и их содержимого.

Верхнее меню на всех страницах:
Кнопка "Домашняя страница" - ссылка на домашнюю страницу

1. Домашняя страница:
-Кнопка "Курсы" - ссылка на страницу списка курсов
-Кнопка "Преподаватели" - ссылка на страницу списка преподавателей
-Кнопка "Учащиеся" - ссылка на страницу списка учащихся
-Кнопка "Компании" - ссылка на страницу списка компаний
-Кнопка "Занятия" - ссылка на страницу списка занятий

2. Страница списка курсов:
-Кнопка "Добавить" - ссылка на страницу добавления (изменения) данных о курсе
-Список курсов:
  Название курса - ссылка на страницу курса
  Название компании

3. Страница курса:
-Название курса
-Название компании
-Дата начала
-Продолжительность курса
-Количество часов в день
-Список преподавателей:
  Имя преподавателя
-Список учащихся:
  Имя учащегося
  Кнопка "Удалить учащегося"
-Кнопка "Расписание занятий" - ссылка на страницу расписания занятий для этого курса
-Кнопка "Изменить" - ссылка на страницу добавления (изменения) данных о курсе
-Кнопка "Добавить учащегося"
-Кнопка "Удалить"

4. Страница расписания занятий для курса:
-Кнопка "Добавить занятие" - ссылка на страницу добавления (изменения) данных о занятии
-Список занятий:
  ID занятия - ссылка на страницу занятия
  Время и дата
  Имя преподавателя

5. Страница списка преподавателей:
-Кнопка "Добавить" - ссылка на страницу добавления (изменения) данных о преподавателе
-Список преподавателей:
  Имя преподавателя - ссылка на страницу преподавателя
  Название компании

6. Страница преподавателя:
-Имя преподавателя
-Название компании
-Список читаемых курсов:
  Название курса
-Кнопка "Расписание занятий" - ссылка на страницу расписания занятий для преподавателя
-Кнопка "Изменить" - ссылка на страницу добавления (изменения) данных о преподавателе
-Кнопка "Удалить"

7. Страница расписания занятий для преподавателя:
Возможна фильтрация по дате (С и ПО)
Список занятий:
-ID занятия - ссылка на страницу занятия
-Название курса
-Дата и время проведения

8. Страница списков учащихся:
-Кнопка "Добавить" - ссылка на страницу добавления (изменения) данных об учащемся
-Список учащихся:
  Имя учащегося - ссылка на страницу учащегося

9. Страница учащегося:
-Имя учащегося
-Список проходимых курсов:
  Название курса
-Кнопка "История обучения" - ссылка на страницу истории обучения учащегося
-Кнопка "Расписание занятий" - ссылка на страницу расписания занятий для учащегося
-Кнопка "Изменить" - ссылка на страницу добавления (изменения) данных об учащемся
-Кнопка "Удалить"

10. Страница расписания занятий для учащегося:
Возможна фильтрация по дате (С и ПО)
Список занятий:
-ID занятия - ссылка на страницу занятия
-Название курса
-Имя преподавателя
-Дата и время проведения

11. Страница списка компаний:
-Кнопка "Добавить" - ссылка на страницу добавления (изменения) данных о компании
-Список компаний:
  Название компании - ссылка на страницу компании

12. Страница компании:
-Название компании
-Адрес
-Список проводимых курсов:
  Название курса
-Список преподавателей:
  Имя преподавателя
-Кнопка "Изменить" - ссылка на страницу добавления (изменения) данных о компании
-Кнопка "Удалить"

13. Страница списка занятий:
-Список занятий:
  ID занятия - ссылка на страницу занятия
  Название курса
  Дата и время проведения

14. Страница занятия:
-ID занятия
-Название курса
-Имя преподавателя
-Дата и время проведения
-Кнопка "Изменить" - ссылка на страницу добавления (изменения) данных о занятии
-Кнопка "Удалить"

15. Страница добавления (изменения) данных о курсе:
-Кнопка "Сохранить"
-Редактируемые поля:
  Название курса
  Название компании
  Дата начала
  Продолжительность
  Количество часов в день

16. Страница добавления (изменения) данных о преподавателе:
-Кнопка "Сохранить"
-Редактируемые поля:
  Имя преподавателя
  Название компании

17. Страница добавления (изменения) данных об учащемся:
-Кнопка "Сохранить"
-Редактируемые поля:
  Имя учащегося

18. Страница добавления (изменения) данных о компании:
-Кнопка "Сохранить"
-Редактируемые поля:
  Название компании
  Адрес

19. Страница добавления (изменения) данных о занятии:
-Кнопка "Сохранить"
-Редактируемые поля:
  Имя преподавателя
  Дата и время проведения

20. Страница истории обучения учащегося:
-Список пройденных курсов:
  Название курса
  Дата начала курса
  Дата окончания (дата начала + продолжительность)
